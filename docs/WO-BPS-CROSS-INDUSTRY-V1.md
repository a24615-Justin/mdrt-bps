# WO-BPS-CROSS-INDUSTRY-V1：異業增員試算工具（房仲/車仲）

> **版本**：v2.0 ｜ **建立日期**：2026-04-09
> **狀態**：規格審查中
> **專案**：mdrt-bps（獨立於保策通）

## 核心定位轉變（v1 → v2）

| | v1（錯的） | v2（對的） |
|--|---------|---------|
| 框架 | 轉行比較：現職 vs 保經 | **加線增收：現職 + 保險 = ?** |
| 話術 | 「來保經賺更多」 | 「你的客戶已經在付保費了，錢不在你口袋而已」 |
| 抗性 | 高（房仲一間賺 30 萬，保險單件 2 萬） | **低（不用放棄任何東西）** |
| 身份 | 放棄現職 → 保險業務 | **現職 + 保險雙軌** |

---

## 功能設計三問

### Q1：誰要用？用在何處？
**保經增員主管**面對房仲、車仲時，用這個工具展示「不用放棄現職，加一條保險收入線能多賺多少」。
場景：① 面談破冰 ② 說明會投影 ③ 傳連結讓對方自己試算

### Q2：目的與功能適配程度？
目的：讓對方看到「**你已經有的客戶池裡藏著多少保費收入，這筆錢現在正流向別人**」。
不是說服他轉行，是讓他意識到正在浪費的機會成本。
現有 BPS 引擎可複用，新增 persona + 客戶池金礦區塊。

### Q3：讓產品更好用，還是更複雜？
更好用。增員主管不用再口頭解釋「為什麼你應該來做保險」，工具直接算給對方看。

## 說服邏輯：三階段遞進

```
Stage 1：額外收入（破冰）
「你的客戶已經在買保險了，加一條線就能多賺 NT$ 14 萬/年」
→ 低抗性，不要求放棄任何東西

Stage 2：被動收入（種子）
「跟賣房/賣車不同，保險有續佣——每年自動進帳，不做事也有」
→ 讓對方第一次聽到「被動收入」這個概念

Stage 3：全職邀請（只在對方主動問時）
「如果你想全力發展，保經的收入天花板比現職高得多」
→ 不主動提，等對方自己好奇
```

**工具只做 Stage 1 + Stage 2。Stage 3 是人的工作，不是工具的工作。**

## 新增內容

### Persona 1：`realtor`（房仲）

#### A. 參數定義

| key | label | type | default | section | 說明 |
|-----|-------|------|---------|---------|------|
| `realtorClients` | 既有客戶/人脈數 | number | 200 | current | 成交+帶看+轉介紹累計 |
| `realtorNewClientsPerYear` | 每年新增客戶 | number | 40 | current | 每年帶看的新客戶數 |
| `realtorAvgPremium` | 客戶平均年保費 | currency | 40000 | opportunity | 房貸壽險+火險+家庭保障 |
| `realtorConvRate` | 預估轉換率 | percent | 15 | opportunity | 既有客戶中願意跟你買保險的比例 |
| `avgProductCommRate` | 平均商品佣金率 | percent | 60 | opportunity | 共用現有參數 |
| `brokerCommRate` | 個人佣金率 | percent | 20 | opportunity | 共用現有參數 |
| `renewalRate` | 續佣率 | percent | 10 | opportunity | 第 2 年起每年續佣佔 FYP 的比例 |

**不需要的參數**（刻意不問）：
- ❌ 房仲年收入（不比較收入，避免「我賣房賺更多」的反彈）
- ❌ 房仲佣金（同上）
- ❌ 波動率（不製造恐懼）

#### B. 收入模型：「加線」邏輯

```
第1年：
  保險客戶 = realtorClients × realtorConvRate
  FYP = 保險客戶 × realtorAvgPremium
  額外收入 = FYP × avgProductCommRate × brokerCommRate
  續佣 = 0

第2年：
  新增客戶 = realtorNewClientsPerYear × realtorConvRate
  新 FYP = 新增客戶 × realtorAvgPremium
  額外收入 = 新 FYP × 佣金率
  續佣 = 第1年 FYP × renewalRate × brokerCommRate

第N年：
  額外收入 = 新 FYP 佣金 + Σ(歷年續佣堆疊)

展示方式：不顯示「現職收入」，只顯示「保險額外收入」
```

#### C. 客戶池金礦試算（核心區塊）

```
┌─────────────────────────────────────────────────┐
│  💎 您的客戶池裡藏著多少保費？                        │
├─────────────────────────────────────────────────┤
│                                                  │
│  您的人脈池         200 人                         │
│  × 預估轉換率        15%                          │
│  = 潛在保險客戶       30 人                         │
│  × 平均年保費     NT$ 4 萬                         │
│  ─────────────────────────────                   │
│  潛在年保費總額   NT$ 120 萬                        │
│                                                  │
│  您的額外收入：                                     │
│  首年佣金        NT$ 14.4 萬                       │
│  第3年起每年續佣   NT$ 7.2 萬（自動）                │
│  第5年起每年續佣   NT$ 12 萬（自動）                  │
│                                                  │
│  📌 這筆保費目前正在流向別的保險業務員。                │
│     您不需要放棄賣房，只需要順手多做一步。              │
│                                                  │
└─────────────────────────────────────────────────┘
```

#### D. 5 年額外收入成長圖

```
額外收入（保險線）

NT$30萬 ┤                                    ████
NT$25萬 ┤                              ████  ████
NT$20萬 ┤                        ████  ████  ████
NT$15萬 ┤                  ████  ████  ████  ████
NT$10萬 ┤            ▓▓▓▓  ▓▓▓▓  ▓▓▓▓  ▓▓▓▓  ▓▓▓▓  ← 續佣（被動）
NT$ 5萬 ┤      ████  ████  ████  ████  ████  ████  ← 新單佣金
        └──────┬─────┬─────┬─────┬─────┬─────
              第1年  第2年  第3年  第4年  第5年

「房子照賣，保險額外賺。續佣那條線，永遠下不來。」
```

**不顯示「現職收入」對比**——避免引發「保險賺比房子少」的心理抗性。

#### E. 心理腳本（3 層，加線敘事）

**第一層｜順手多賺的錢（Stage 1 破冰）**
> 您每成交一間房子，買方接下來會做什麼？辦房貸、買火險、規劃家庭保障。這些事他本來就要做，差別只在於：是跟陌生的保險業務員做，還是跟他信任的您做。如果是您來服務，每位客戶平均每年 NT$ 4 萬保費，您可以額外拿到約 NT$ 4,800 的佣金——不用多花時間找客戶，因為人就在您面前。

**第二層｜下不來的地板（Stage 2 種子）**
> 賣房最辛苦的是什麼？每個月歸零重新開始。保險有一個房仲沒有的東西：**續期佣金**。您今年服務的 30 位保險客戶，明年每一位續保，您自動收到佣金。後年還是。這筆錢不需要您做任何事——它是您過去服務的「利息」。三年後，光續佣就有每年 NT$ 7 萬自動進帳。

**第三層｜同一群人，兩份收入（總結）**
> 您不需要「轉行」做保險。您繼續做最擅長的房仲，只是在每次成交後多問一句：「房貸壽險和火險要不要一起處理？」這一句話的價值，五年後累計超過 NT$ 100 萬。這 100 萬本來就存在——差別只在於進誰的口袋。

### Persona 2：`auto`（車仲）

#### A. 參數定義

| key | label | type | default | section | 說明 |
|-----|-------|------|---------|---------|------|
| `autoClients` | 歷年交車客戶數 | number | 150 | current | 累計交車名單 |
| `autoNewUnitsPerYear` | 每年新交車數 | number | 48 | current | 月均 4 台 × 12 |
| `autoAvgPremium` | 客戶平均年保費 | currency | 35000 | opportunity | 車險+壽險+意外險 |
| `autoConvRate` | 預估轉換率 | percent | 20 | opportunity | 車主 → 保險客戶（高，因車險是剛需） |

**不需要的參數**：
- ❌ 底薪/獎金（不比較收入）
- ❌ 獎金遞減（不製造恐懼）

#### B. 客戶池金礦試算

```
┌─────────────────────────────────────────────────┐
│  💎 您的交車名單裡藏著多少保費？                      │
├─────────────────────────────────────────────────┤
│                                                  │
│  歷年交車客戶      150 人                          │
│  × 預估轉換率       20%                           │
│  = 潛在保險客戶      30 人                          │
│  × 平均年保費    NT$ 3.5 萬                        │
│  ─────────────────────────────                   │
│  潛在年保費總額  NT$ 105 萬                         │
│  您的額外佣金    NT$ 12.6 萬/年                     │
│                                                  │
│  🚗 每台新車 = 一個保險觸及機會                       │
│     車險是最自然的切入點，不需要「推銷」               │
│     每年交車 48 台 = 48 次自然開口機會                │
│                                                  │
│  📌 這筆保費目前正在流向別的保險業務員。                │
│                                                  │
└─────────────────────────────────────────────────┘
```

#### C. 心理腳本（3 層，加線敘事）

**第一層｜每台車背後的保費（Stage 1 破冰）**
> 您每交一台車，車主接下來一定要做什麼？保強制險、買任意險、可能還需要壽險和意外險。這些事 100% 會發生。差別只在於：這筆保費是流向某個不認識車主的業務員，還是流向車主最信任的您。每位車主平均年保費 NT$ 3.5 萬，您經手的話每年額外收入約 NT$ 4,200。48 台車就是 NT$ 20 萬。

**第二層｜賣完車才是開始（Stage 2 種子）**
> 賣車跟賣保險最大的差別：車子交了就結束，保險交了才開始。您今年服務的 30 位保險客戶，明年續保時您自動收到續佣——不用打電話、不用約見面、不用重新成交。這筆被動收入每年疊加：第三年起每年 NT$ 6 萬自動進帳，第五年 NT$ 10 萬。車照賣，續佣照收。

**第三層｜最自然的開口（總結）**
> 交車那一刻，車主心情最好、對您信任最高。這時候說一句：「車險我幫您一起處理好，順便看一下意外險有沒有需要加強？」——這不是推銷，這是**服務的延伸**。您不需要轉行，不需要辭職，只需要在交車流程裡多加 15 分鐘。五年後，這 15 分鐘的累計價值超過 NT$ 80 萬。

## 合規注意事項

- 所有數字標示「僅供參考，實際收入依個人表現與市場狀況而定」
- 不使用「保證收入」「一定賺更多」等處方性語言
- 不貶低對方的現職（「房仲不好」「車仲沒前途」→ 絕對禁止）
- 心理腳本語氣：「多一條線」「順手多賺」「不用放棄」
- 轉換率標示「預估值，實際依個人客戶關係品質而異」
